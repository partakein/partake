OpenID でログイン

Twitter が落ちている場合などに、Open ID でもログインすることができます。

Twitter をメッセージ基盤として利用するため、Twitter アカウントとの結びつけが必要です。 Twitter ID と Open ID の結び付け設定は済んでいない場合、Twitter でログインしてから設定してください。
